HTML

  • URL is an acronym for Uniform Resource Locator
  • HTML is an acronym for HyperText Markup Language
  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • There are 3 types of lists used in HTML Unorderd list, Ordered List (numbered), & List Items

CSS

  • CSS is an acronym for Cascading Style Sheets.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Adding a percentange to an element will assign the percentage given of the available space referenced in the selector.
  • A class selector in an HTML element assigned to the "class" i.e. (.card, Check DevTools in Chrome) will gain these property assignments from the CSS.

Git Commands

  • cd: Choose Directory on GitBash - navigate through folder structure.
  • mkdir: make a new folder/directory, never use spaces when making a directory.
  • git checkout -b (branch-name): creates a new branch and switches to that directory.
  • ls ../ : shows all folders in a main directory, which is important to find you main git directory.
  • git push origin main: [push files to GitHub] add new files and changes that were created on Visual Studio Code.

JavaScript

  • A variable is a named container that allows us to store dta in our code.
  • There are two requirements for a JavaScript function which are 'Defining a Function' & 'Calling a Function.'
  • The function Math.random() generates a random integer value greater than 0 to and integer value > 1, i.e. all values between 0 & 1.
  • Control flow is the order in which a computer executes code in a script, in JavaScript it is in decending order so ORDER MATTERS!
  • Defining a fuction entials that we create a specific task we want our code to run & calling a function is used to execute the function when needed.
