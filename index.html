Git Commands

  • cd: Choose Directory on GitBash - navigate through folder structure
  • git checkout -b (branch-name): creates a new branch and switches to that directory
  • mkdir: make a new folder/directory, never use spaces when making a directory
  • ls ../ : shows all folders in a main directory, which is important to find you main git directory
  • git push origin main: [push files to GitHub] add new files and changes that were created on Visual Studio Code
